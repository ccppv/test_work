# Business Dashboard — Портфельный дашборд

Мини-дашборд для анализа бизнес-метрик за 12 месяцев с AI-инсайтами от **Claude** (Anthropic) и **Gemini** (Google).

---

## Запуск локально

```bash
# 1. Клонировать репозиторий
git clone <repo-url>
cd business-dashboard

# 2. Установить зависимости
npm install

# 3. Создать .env.local и добавить API-ключи
cp .env.example .env.local
# Отредактируйте .env.local — вставьте ключи Anthropic и/или Google

# 4. Запустить dev-сервер
npm run dev
# Открыть http://localhost:3001
```

## Как добавить API-ключи

### Claude (Anthropic)
1. Получить ключ на https://console.anthropic.com
2. Добавить в `.env.local`:
   ```
   ANTHROPIC_API_KEY=sk-ant-...
   ```

### Gemini (Google)
1. Получить бесплатный ключ на https://aistudio.google.com/app/apikey
2. Добавить в `.env.local`:
   ```
   GEMINI_API_KEY=AIza...
   ```

Оба ключа используются **только на сервере** через `/api/insight` — во frontend не попадают.  
Провайдера можно переключить кнопками в блоке AI-инсайта.

**На Vercel:** добавить `ANTHROPIC_API_KEY` и `GEMINI_API_KEY` в Settings → Environment Variables

## Продуктовые решения

- **KPI-карточки**: Выручка, Новые клиенты, Маржа, CAC — 4 метрики, которые сразу дают «здоровье» бизнеса
- **Маржа + CAC + Отток на одном графике**: ключевые тревожные сигналы видны вместе
- **Автоматические аномалии**: алгоритм подсвечивает просадки (CAC +20%, маржа -3 п.п., отток +1 п.п., LTV -5%, выручка -5%)
- **Фильтр по периоду**: клик по месяцам сдвигает границы диапазона

## Логика AI-инсайта

**Файл:** `app/api/insight/route.ts`

Клиент отправляет `POST /api/insight` с данными за выбранный период. Сервер форматирует данные в текстовую таблицу с конкретными числами и передаёт в Claude.

### Системный промпт

```
Ты — финансовый аналитик портфельной компании. Тебе предоставлены бизнес-метрики за выбранный период.

Твоя задача:
1. Выдели 3–5 ключевых наблюдений, опираясь ТОЛЬКО на конкретные цифры из таблицы
2. Укажи тревожные сигналы: рост оттока, падение маржи, рост CAC, снижение LTV
3. Отметь позитивную динамику там, где она есть
4. Сформулируй 1–2 конкретных вопроса, которые стоит проверить в первую очередь

Требования к ответу:
- Короткие пункты с конкретными цифрами — без воды и общих фраз
- Не выдумывай факты, которых нет в данных
- Если данных недостаточно для уверенного вывода — так и напиши
- Ответ на русском языке
```

**Почему так**: Промпт задаёт роль, явно запрещает выдумывать факты и требует цифр — снижает галлюцинации. Структура «наблюдения + тревоги + позитив + вопросы» даёт управленческий вывод.
